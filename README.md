# AI-Powered Smart Mail Reminder Assistant

## Project Overview

Smart Mail Reminder Assistant is a Python-based automation project that monitors Gmail inboxes and sends instant Telegram notifications for important emails.

The system automatically detects emails related to internships, placements, interviews, deadlines, jobs, hackathons, and career opportunities. It works even when the user's laptop is turned off by running through GitHub Actions.

---

## Problem Statement

Many students miss important emails because they do not check their inbox regularly.

Missing emails related to:

- Internships
- Placements
- Interviews
- Hackathons
- Deadlines
- Career Opportunities

can lead to missed opportunities.

This project solves that problem by automatically monitoring Gmail and sending alerts directly to Telegram.

## Project Workflow

1. Authenticate Gmail Account
2. Read Unread Emails
3. Check Email Subject
4. Match Important Keywords
5. Generate Alert
6. Send Notification To Telegram
7. Repeat Automatically Every 15 Minutes
